Guia de comandos

Una vez instalado GIT procederemos a su configuración con los siguientes Comandos

Comandos de configuración

  1. git --version (Nos muestra la versión de GIT instalada)
  2. git config --global --list (Este comando nos muestra la configuración actual. Para salir de la visualización presionar la tecla "Q")
  3. git config --global user.name nombre (Este comando nos permite configurar el nombre con el que se subirán los datos al Repositorio)
  4. git config --global user.email tucorreo@gmail.com (Este comando nos permite configurar el correo, debe ser el mismo con el que se creó el Repositorio)
  5. git config --global core.editor "code --wait" (Este comando configurará VS code como el editor de archivos de configuración de GIT)
  6. git config --global -e (Este comando abre el archivo de configuración en VS code)
